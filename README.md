# ZeroClaw Skill

Agent skill for [ZeroClaw](https://github.com/zeroclaw-labs/zeroclaw), the Rust runtime for autonomous AI assistants.

## Install

```bash
npx skills add Perseusmx/zeroclaw-skill
```

Or clone manually:

```bash
git clone https://github.com/Perseusmx/zeroclaw-skill.git
```

## What's Included

- Current Zeroclaw install, onboarding, update, and troubleshooting guidance
- Provider reference aligned to the upstream docs catalog
- Channel reference aligned to the currently documented channel set
- Configuration and security notes for supervised deployments
