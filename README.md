# ZeroClaw Skill

Agent skill for [ZeroClaw](https://github.com/zeroclaw-labs/zeroclaw) — fast, small, fully autonomous AI assistant infrastructure built in Rust.

## Install

```bash
npx skills add Perseusmx/zeroclaw-skill
```

Or clone manually:

```bash
git clone https://github.com/Perseusmx/zeroclaw-skill.git
```

## What's Included

- Installation, onboarding, and essential CLI commands
- 10+ built-in AI providers plus custom endpoint support
- 16+ communication channels (Telegram, Discord, Slack, WhatsApp, Signal, Matrix, iMessage, and more)
- Complete configuration reference
- Security best practices and deployment checklists

## Usage

After installation, the skill is automatically available to compatible AI agents. Agents will use it when they detect tasks related to ZeroClaw setup, configuration, providers, channels, or troubleshooting.

For complete documentation, see the [ZeroClaw project](https://github.com/zeroclaw-labs/zeroclaw).
